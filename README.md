# luniverse-atom-plugin package

## Test & Contribution

```
$ git clone https://github.com/Lambda256/luniverse-atom-plugin.git
$ cd luniverse-atom-plugin
$ npm install
$ apm link

Atom에서 [Cmd+Shift+P]로 Window: Reload를 한 후에 Packages > Luniverse-Atom-Plugin을 확인
```
